# 민족통일협의회 영동군 지부 웹사이트

## 프로젝트 소개
민족통일협의회 영동군 지부의 공식 웹사이트 프로젝트입니다.
통일운동, 민족화합, 지역사회 활동을 소개하고 회원들과 소통하는 플랫폼입니다.

## 프로젝트 구조

```
MintongYeongdong/
├── index.html          # 메인 홈페이지 (조직 소개, 비전, 주요 서비스)
├── about.html          # 협의회 소개 (설립 목적, 역사, 미션)
├── services.html       # 사업 안내 (통일지도력 양성, 민족화합 등)
├── members.html        # 조직 안내 (지도부, 핵심 회원)
├── news.html           # 활동 소식 (뉴스, 공지사항, 블로그)
├── portfolio.html      # 활동 갤러리 (프로젝트 사진, 성과)
├── casestudy.html      # 성과 사례 (구체적인 프로젝트 분석)
├── events.html         # 행사 안내 (세미나, 강연, 토론회)
├── contact.html        # 연락처 (오시는 길, 문의하기)
├── assets/             # CSS, JS, 이미지 등 리소스
└── vendors/            # Bootstrap, Swiper 등 외부 라이브러리
```

## 페이지 설명

### 1. index.html - 메인 홈페이지
- 조직의 비전과 미션 소개
- 주요 서비스 요약
- 팀 멤버 소개
- 프로세스 안내 (계획→디자인→개발→배포)
- 가격 정책 / 회원가입 안내
- 성과 통계 (회원 수, 프로젝트 수 등)

### 2. about.html - 협의회 소개
- 설립 목적 및 배경
- 조직의 역사와 연혁
- 통일운동의 비전과 가치
- 주요 성과 통계
- 조직 문화

### 3. services.html - 사업 안내
**주요 사업 영역:**
- 통일지도력 양성 프로그램
- 민족화합 사업
- 통일대비 사업
- 국민교육 및 안보사업
- 출판 및 홍보활동

### 4. members.html - 조직 안내
- 지도부 소개 (회장, 부회장, 이사진)
- 핵심 회원 프로필
- 지역 인플루언서
- 회원 약력 및 전문분야
- 소셜미디어 링크

### 5. news.html - 활동 소식
- 최신 뉴스 및 공지사항
- 통일 정책 논평
- 세미나/토론회 후기
- 회원 기고문
- 카테고리: 통일정책, 민족화합, 교육프로그램, 행사소식

### 6. portfolio.html - 활동 갤러리
- 통일교육 프로그램 사진
- 남북교류 활동
- 지역사회 활동
- 청년 참여 프로그램
- 프로젝트 갤러리

### 7. casestudy.html - 성과 사례
- 성공 프로젝트 심층 분석
- 구체적인 성과 지표
- 지역사회 영향력
- 베스트 프랙티스

### 8. events.html - 행사 안내
- 정기 세미나 및 토론회
- 통일 강연회
- 민족화합 행사
- 이벤트 갤러리 슬라이더
- 행사 일정 및 참가 신청

### 9. contact.html - 연락처
- 영동군 사무소 위치
- Google Maps 통합
- 연락처 (전화, 이메일)
- 문의 양식
- 회원가입 / 후원 문의
- 소셜미디어 링크

## 기술 스택
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript
- GSAP (애니메이션)

## 로컬 테스트 방법

### 방법 1: 직접 브라우저로 열기
```
index.html 파일을 더블클릭하여 브라우저에서 열기
```

### 방법 2: Live Server 사용 (권장)
VS Code의 Live Server 확장 프로그램 사용
```
1. VS Code에서 프로젝트 폴더 열기
2. index.html 우클릭
3. "Open with Live Server" 선택
```

### 방법 3: Python HTTP 서버
```bash
cd C:\yd\MintongYeongdong
python -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```
